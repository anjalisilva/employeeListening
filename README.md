# Aerodyne Employee Listening and Job Architecture Analysis

A reproducible R workforce-analytics project examining where employees experience structural career ceilings and how those conditions are associated with career-growth sentiment, recent promotion, and voluntary turnover. The analysis combines employee records, job architecture, and listening-survey data; validates the career-growth measure; evaluates adjusted relationships; assesses equity exposure; and tests the sensitivity of key assumptions.

## Dataset
